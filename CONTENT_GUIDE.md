# Homepage Content Guide

This homepage is now maintained through three primary files:

- `_data/home.yml`: edit homepage content, including profile links, introduction, statistics, news, research focus, publications, education, honors, service, teaching, and internships.
- `assets/css/site.css`: edit visual design, colors, spacing, typography, cards, responsive behavior, and dark mode.
- `index.html`: edit page structure, section order, navigation anchors, and how content from `_data/home.yml` is rendered.

The old academic template files are still in the repository for reference, but the live homepage no longer depends on `_pages/about.md`, `_layouts/default.html`, or `assets/css/main.scss`.
